## Commands

The commands below require `manage_server` permission.

| Command             | Example               | Description                                     |
| :------------------ | :-------------------- | :---------------------------------------------- |
| `?gcreate`          | `?gcreate`            | Create a giveaway in an interactive way         |
| `?gstart `          | `?gstart`             | Start a giveaway with a specific                |
| `?greroll <msg_id>` | `?greroll 1234567890` | Reroll a specific giveaway with that message id |
| `?gend <msg_id>`    | `?gend 1234567890`    | End a specific giveaway with that message id    |
| `?gdelete <msg_id>` | `?gdelete 1234567890` | Delete a specific giveaway with that message id |
